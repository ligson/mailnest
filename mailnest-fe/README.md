# Mail Nest 前端

前端使用 Vue 3 + Vite + TypeScript + Ant Design Vue 开发，当前已实现：

- 登录页。
- 注册页。
- 基础应用壳。
- Pinia 登录态管理。
- 统一 envelope API 客户端。
- 邮箱账号列表、新增和删除页面。
- 邮箱连接测试和手动收取按钮。
- 邮件列表和邮件详情抽屉。
- Microsoft OAuth2 授权入口和回调页面。
- 开发环境 `/api` 代理到后端 `http://127.0.0.1:8080`。

## 启动

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:5173
```

## 构建

```bash
npm run build
```
